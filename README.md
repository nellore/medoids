medoids
=======

Code for simulations and figures of the paper "Recovery guarantees for exemplar-based clustering," available at http://arxiv.org/abs/1309.3256 . Simulations were done in MATLAB using the Gurobi Optimizer 5.5 API. Figures were generated in Mathematica. randsphere.m is by Roger Stafford, and sqdistance.m is by Michael Chen.
